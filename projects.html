Projects

Throughout my time coding, I have developed some projects that I have displayed below. They mainly utilize Javascript and its frameworks, with some other languages for additional leverage.

Pokemon Music Listener

Description

Pokemon Music Listener is an application made to listen to famous music tracks in the Pokemon Franchise. It's programmed to accept simple click and key inputs, and these event listeners were made with jQuery.

Technologies

  • HTML
  • CSS
  • jQuery
Neighbourhood Watch

Description

Neighbourhood Watch is a app built using both client-side and server-side JavaScript. Its purpose is to give users a platform to report recent events that take place in their community. Posts made on the website are visible on the posts page, and there is a built in mapping system where the location of the post is inputted in.

Technologies

  • ReactJS
  • ExpressJS
  • PostgreSQL
  • CSS
React Counter

Description

React Counter is a program built to do basic arithmetic with an easy to use interface. There is a drop down menu that's used to select the desired operation, and buttons that allow you to either increase or decrease the value of the current number.

Technologies

  • ReactJS
  • CSS
